Examination Guidelines

Exam Rules & Instructions

Read all rules carefully before beginning the examination

  • 01Total Questions: 100 Multiple Choice Questions covering all RBI cyber security awareness domains.
  • 02Time Limit: 90 Minutes. Exam auto-submits when the countdown reaches zero.
  • 03Format: Each question has four options (A, B, C, D). Select the single correct answer.
  • 04Navigation: Move freely between questions using Previous/Next or the Question Navigator panel.
  • 05Passing Marks: Minimum of 70 out of 100 (70%) is required to pass.
  • 06Difficulty: 25 Easy · 50 Medium · 25 Hard questions, randomised order each attempt.
  • 07Randomisation: Both question order and option order are shuffled every new attempt.
  • 08Results: Score, percentage, correct/wrong counts, and time used shown immediately after submission.
  • 09Review Mode: After results, review all questions with correct answers and explanations.
  • 10Practice Mode: No time limit; correct answer highlights and blinks immediately after each selection.
⚠️ Important: This simulator is designed to prepare bank employees for RBI cyber security awareness assessments. All content is based on publicly available RBI guidelines and banking IT security best practices.
